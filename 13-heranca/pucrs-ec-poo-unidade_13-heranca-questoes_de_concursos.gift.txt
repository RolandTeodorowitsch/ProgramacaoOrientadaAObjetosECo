::POO - Herança - Questões de Concurso - 003::[html]<p align="justify"><b>(UFG - 2023 - Técnico de Tecnologia da Informação - Instituto Verbena - 36)</b> No contexto de programação orientada a objetos, suponha três classes denominadas A, B e C. 	~A herda de B.#
	~A herda de C.#

::POO - Herança - Questões de Concursos - 004::[html]<p align="justify"><b>(INMETRO - 2014 - Assistente Executivo em Metrologia e Qualidade / Informática - IDECAN - 60)</b> A linguagem de programação C++, uma evolução da linguagem de programação C, pode ser considerada uma linguagem híbrida, pois é possível utilizar a programação estruturada e a orientada a objetos. Os programas na linguagem C++ consistem em peças‐chave denominadas classes e funções. Ao derivar uma classe a partir de uma classe base, esta pode ser herdada como <code>public</code>, <code>protected</code> ou <code>private</code>. "Ao derivar uma classe a partir de uma classe 
base com herança <code>protected</code>, os membros <code>public</code> da classe base tornam‐se membros ___________ da classe derivada 
e os membros <code>protected</code> da classe base tornam‐se membros ___________ da classe derivada." Assinale a alternativa que completa correta e sequencialmente a afirmativa anterior.</p>{
	~<code>public</code> / <code>private</code>#
	~<code>private</code> / <code>private</code>#
	~<code>public</code> / <code>protected</code>#
	~<code>protected</code> / <code>public</code>#
	=<code>protected</code> / <code>protected</code>#
}

::POO - Herança - Questões de Concursos - 005::[html]<p align="justify"><b>(TCE/AC - 2009 - Analista de Controle Externo / Processamento de Dados - CESPE - 80)</b> Com relação às linguagens de programação orientadas a objetos, assinale a opção correta.</p>{
	~Um operador de conversão <code>static</code>, ou operador de coerção, pode ser utilizado para se converter um objeto de uma classe em um objeto de outra classe, exceto quando se tratar de um objeto de tipo fundamental.#
	~No C++, na herança entre classes, quando a derivação da classe base é do tipo <code>public</code>, os membros <code>public</code> e <code>protected</code> da classe base tornam-se membros <code>private</code> e <code>protected</code> na classe derivada.#
	~Cada método de uma classe é especificado por uma assinatura cuja composição não inclui a lista de argumentos.#
	=Em C++, uma função operadora global pode tornar-se um <code>friend</code> de uma classe se precisar acessar diretamente membros <code>private</code> ou <code>protected</code> dessa classe.#
	~O modificador de visibilidade não pode ser do tipo <code>public</code>.#
}

::POO - Herança - Questões de Concurso - 006::[html]<p align="justify"><b>(UFG - 2023 - Técnico de Tecnologia da Informação - Instituto Verbena - 31)</b> Em linguagem Java, modificadores de acesso são empregados para definir a visibilidade de acesso. Quando aplicados a membros de classe,</p>{
		~os membros <code>protected</code> na classe C são acessíveis a qualquer superclasse de C.#
		=os membros <code>protected</code> na classe C são acessíveis a qualquer subclasse de C.#
		~os membros <code>private</code> na classe C são acessíveis a qualquer classe do mesmo pacote de C.#
		~os membros <code>private</code> na classe C são acessíveis a qualquer subclasse de C.#
}
